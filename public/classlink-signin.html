How to Login

Extensions Not Allowed (School Devices) ▼

For Chromebooks and Managed School Devices

  1. Click on "Login with ClassLink"
  2. You'll be redirected to this page
  3. Follow the below instructions on another Windows/Mac device
  4. Copy the login code from that device and send it to this device
  5. Click "Submit" to complete your login
Android/iOS (Website) ▼

For Android/iOS (Website)

  1. Click on "Login with ClassLink"
  2. You'll be redirected to this page
  3. Follow the below instructions on another Windows/Mac device
  4. Copy the login code from that device and send it to this device
  5. Click "Submit" to complete your login
Chrome/Edge on Windows/Mac ▼

Install the Gradexis Browser Extension

  1. Download the Gradexis Login extension: Gradexis Extension
  2. Unzip the files into a folder
  3. Go to chrome://extensions (copy and paste it)
  4. Turn on the Developer mode switch
  5. Click Load Unpacked and select the Gradexis Extension
  6. Reload this page to automatically login
